## 설치
1. python `virtualenv`를 생성하고 활성화합니다.
    ```bash
    $ python -m virtualenv .venv
    $ . .venv/bin/activate
    ```

2. `requirements.txt`를 이용해 필요 패키지를 설치합니다.
    ```bash
    (.venv) $ pip install -r requirements.txt
    ```

3. `app.py`를 실행합니다.
    ```bash
    (.venv) $ python app.py
    ```
